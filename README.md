# OracleForms
Exemplo simples de uso do Oracle Forms

Roteiro para uso:
1) executar o script create.sql: Sera criado o schema SISTEMA com a senha SISTEMA e os objetos de banco de dados;
2) executar o script testes.sql: Vai criar massa de dados e executar testes unitários;
3) Descompactar arquivo botoes.zip: arquivo com as imagens dos botões da toolbar;
4) Abrir o arquivo frmClientes.fmb no Oracle Forms ou executar o frmClientes.fmx diretamente pelo runtime;
5) executar o script drop.sql: Remove o schema SISTEMA do banco de dados;
